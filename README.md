# PIE ML - Frontend Angular

Interface citoyen de la Plateforme d'Immatriculation des Engins du Mali.

## Prérequis

- Node.js 20+
- Backend Spring Boot en cours d'exécution

## Démarrage

```bash
npm install
npm start
```

Application disponible sur `http://localhost:4200`

## Fonctionnalités

- Page d'accueil (landing) conforme à la maquette
- Inscription avec validation OTP
- Connexion JWT
- Tableau de bord citoyen
- Workflow en 4 étapes : engin → documents → paiement → rendez-vous
- Suivi de dossier public
